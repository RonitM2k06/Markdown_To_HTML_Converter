# Project Structure

```
MarkdownToHtml/
├── src/
│   ├── config/       # Settings and constants
│   ├── controllers/  # Logic mediators
│   ├── models/       # Data representation
│   ├── rendering/    # HTML and Markdown parsers
│   ├── services/     # File and auto-save handling
│   ├── ui/           # PySide6 components
│   ├── utils/        # Helpers and logging setup
│   └── main.py       # Entry point
├── tests/            # Pytest test cases
├── assets/           # Icons and themes
├── docs/             # Documentation
└── requirements.txt
```
